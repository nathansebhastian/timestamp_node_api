API PROJECT: Timestamp Microservice

Pass a string as a parameter, and the API will check to see if it is a date.

The string can be a natural language date (for example, January 25, 2016) or a Unix timestamp.

The API will then return a JSON object with both the Unix timestamp and the natural language date.

You can try these three links for example:

https://timestamp-node-api.herokuapp.com/January%2025%202016 = {"unix":"1453680000","natural":"January 25th, 2016"}
https://timestamp-node-api.herokuapp.com/gibber = {"unix":null,"natural":null}
https://timestamp-node-api.herokuapp.com/1236038400 = {"unix":"1236038400","natural":"March 3rd, 2009"}
